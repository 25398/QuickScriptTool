先跑鼠大侠

启动窗口模式宏后，本机桥会在 127.0.0.1:19228+ 监听。

02

扩展装在游戏同一用户配置

在游戏那个 Edge 窗口打开 edge://extensions，加载 / 重载本扩展（版本须为 1.1.22+）。跑宏时若仍显示「等待」，点右侧「重新连接」。

顶部黄条「已开始调试此浏览器」无法由扩展关闭，也不能限定到单个标签。可在 Edge 快捷方式追加 --silent-debugger-extension-api（完全退出 Edge 后生效）。

03

状态变为已连接 / 已附着

成功时会出现 connected 或 attached，并带上游戏页标题。
