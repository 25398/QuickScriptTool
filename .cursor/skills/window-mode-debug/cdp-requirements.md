# CDP / 配套扩展窗口模式 — 需求文档（用户约定）

来源：用户联调约定（2026-07）。踩坑：[cdp-lessons.md](cdp-lessons.md)。  
代码：`src/window_mode/window_mode_requirements.h`。

---

## 1. 角色分工

| 角色 | 允许做 | 禁止做 |
|------|--------|--------|
| **配套扩展** | 键鼠、找图、保活、attach | — |
| **Win32（CDP）** | Minimize→Move「鼠标宏」→ **Pin+屏外（工作区尺寸）**；观看 UnPin+迁回宏桌面+还原 | 异桌裸还原；运行期 Move；切桌面钉视；对已展开窗再 SetWindowPlacement |
| **softMessage** | Move「鼠标宏」 | 运行期 GoTo |

一句话：**扩展执行；Pin+屏外防切屏且出帧；你自己进宏桌面后再 UnPin 展开。**

---

## 2. 零切屏（硬）

1. **禁止异桌裸 SoftRestore**（无 Pin）：一点鼠标就会把用户切到宏桌面。  
2. Pin 后再还原+屏外：不切虚拟桌面。  
3. 观看泵 dwell≈2s；**已展开则完全不要碰窗**。  
4. 禁止运行期 GoTo 钉桌泵。

---

## 3. 可展开 + 准点

1. 屏外时用**工作区宽高**，避免缩窗导致 iframe 点偏。  
2. 用户进「鼠标宏」dwell 后：UnPin → Move 宏桌面 → 还原屏上坐标。  
3. 离开：再 Pin+屏外（禁最小化冻帧）。  
4. 结束：UnPin + 恢复坐标 + 最小化。

## 5. 验收

| 现象 | 期望 |
|------|------|
| 跑宏 | 不自动跳宏桌面；鼠标后也不跳 |
| 进宏桌面 | ≈2s 可展开 |
| 找图/点击 | live；点位稳定 |
| 结束 | 可从任务栏展开 |
